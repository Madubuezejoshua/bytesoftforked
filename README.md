## How can I edit this code?

**Use your preferred IDE**

Clone this repo and work locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

**Edit a file directly in GitHub**

- Navigate to the desired file(s)
- Click the "Edit" button (pencil icon) at the top right
- Make your changes and commit

**Use GitHub Codespaces**

- Click the "Code" button on the main repository page
- Select the "Codespaces" tab
- Click "New codespace" to launch
- Edit files and commit your changes

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Firebase (for authentication and database)
- LiveKit (for video conferencing)

## How can I deploy this project?

This project is deployed on Vercel. Push changes to your main branch and they will be automatically deployed.

## Building for Production

```sh
npm run build
```
